## How to contribute

1. Jeśli używasz powershell użyj 
```pwsh
 $ Set-ExecutionPolicy Unrestricted -Scope Process
 ```
 2. Następnie uruchom skrypt "activate" znajdujący się w folderze L.E.S.Z.E.K
 ```pwsh
 $ .\leszek\Scripts\activate
 ```
